# AlertDialog

AlertDialog.Builder builder = new AlertDialog.Builder(context); 构建Dialog的各种参数

*   setIcon 添加Icon
*   setTitle 添加标题
*   setMessage 添加消息
*   setView 设置自定义布局
*   create 创建Dialog
*   show 显示对话框
*   setPositiveButton 确定按钮
*   setNegativeButton 取消按钮
*   setNeutralButton 中间按钮
