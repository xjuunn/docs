## 安装

>   https://pan.baidu.com/s/1LHbEvY6aZ8pG8IbVDyvAsg?pwd=fduy

*   安装**DOSBox0.74**
*   把MASM目录放到C盘外的其他盘根目录

## 挂载

~~~ shell
mount c E:masm
c:
~~~

## Debug 使用

~~~ shell
debug
~~~

-   r    查看CPU寄存器   r ax 修改寄存器的值   read
-   d    查看内存内容   d 1000:0000    f 限制输出数量   du
-   e    修改内存内容   e 0000:0000 22 23 44 55    edit
-   u    将内存的机器码翻译成汇编指令   
-   t     执行一条机器指令   cs:ip 对应的地址    
-   a    以汇编的格式写入内存一条指令    add
