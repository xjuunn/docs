# CSS3 3D转换

# **rotateX()**

围绕x轴旋转

```css
div
{
    transform: rotateX(120deg);
    -webkit-transform: rotateX(120deg); /* Safari 与 Chrome */
}
```

# rotateY()

围绕Y轴旋转

```css
div
{
    transform: rotateY(130deg);
    -webkit-transform: rotateY(130deg); /* Safari 与 Chrome */
}
```
