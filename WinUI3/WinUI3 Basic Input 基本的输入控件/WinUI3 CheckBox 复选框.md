# 有三个选项的复选框

```xml
<CheckBox IsThreeState="True" Content="有三个选项的复选框"/>
```

# 属性

- IsThreeState 是否有三种状态
- IsChecked 是否勾选

# 方法

- Checked 被选中

- Unchecked 被取消勾选
