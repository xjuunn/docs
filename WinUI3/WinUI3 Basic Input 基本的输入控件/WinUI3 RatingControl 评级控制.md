# WinUI3 RatingControl 评级控制

```xml
<RatingControl AutomationProperties.Name="Simple RatingControl" IsClearEnabled="True"
    IsReadOnly="False" Caption="Your rating"/>
```

# 属性

- PlaceholderValue 当前分数
- IsClearEnabled 点击清空
- IsReadOnly 只读
- Caption 提示文本
