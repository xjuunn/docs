# CSS3 文字效果

- text-shadow
- box-shadow
- text-overflow
- word-wrap
- word-break

# 文本阴影

```css
text-shadow:5px 5px 5px #FF0000; 
```

# 盒子阴影

```css
box-shadow:10px 10px 5px #888888;
```

# 溢出

```css
text-overflow:ellipsis;
```

# 换行

```css
word-wrap:break-word;
```
